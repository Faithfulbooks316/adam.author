# Adam Joseph Hanrahan — Author Website

Personal author site for **Adam Joseph Hanrahan**, published by **FaithfulBooks Publishing**.

## Files

| File | Purpose |
|---|---|
| `index.html` | Main site page |
| `styles.css` | Site stylesheet |
| `app.js` | Site JavaScript — add your code here |
| `netlify.toml` | Netlify deployment configuration |

## Deploying to Netlify

1. Push this repository to GitHub (or connect it in the Netlify dashboard).
2. In Netlify → **Add new site → Import an existing project** → pick this repo.
3. Build settings are handled automatically by `netlify.toml` (no build command needed — static site).
4. Click **Deploy site** — Netlify will publish `index.html` from the root folder.

## Local preview

Open `index.html` directly in your browser, or use a simple local server:

```bash
npx serve .
```
